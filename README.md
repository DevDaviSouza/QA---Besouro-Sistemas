# 🧪 QA - Besouro Sistemas

Repositório criado para documentar e demonstrar práticas de Quality Assurance aplicadas ao sistema Besouro Sistemas.

O objetivo deste projeto é centralizar:
- planejamento de testes
- suítes de testes
- casos de testes
- execuções
- bug reports
- automação
- evidências

---

# 🎯 Objetivo

Garantir a qualidade, estabilidade e confiabilidade do sistema através de processos estruturados de QA, utilizando testes manuais e automatizados.

# 🧩 Estrutura de QA Utilizada

```text

 └── Suíte de Teste
      └── Caso de Teste
           └── Execução
                └── Bug Report
```

Os planos de testes documentam:
- escopo
- objetivos

